---
title: 'Get Dataset'
description: 'Retrieve a single dataset by ID'
api: 'GET /datasets/{id}'
---

## Endpoint

```
GET https://api.rowbase.com/v1/datasets/{id}
```

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The dataset ID (e.g., `ds_abc123`) |

## Response

```json
{
  "data": {
    "id": "ds_abc123",
    "name": "Customer List",
    "project_id": "proj_xyz",
    "row_count": 1250,
    "column_count": 8,
    "columns": [
      {"name": "email", "type": "text"},
      {"name": "name", "type": "text"},
      {"name": "created_at", "type": "date"},
      {"name": "plan", "type": "text"},
      {"name": "mrr", "type": "number"},
      {"name": "active", "type": "boolean"}
    ],
    "primary_key": ["email"],
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-20T14:22:00Z"
  }
}
```

## Example Request

<Tabs>
  <Tab title="cURL">
    ```bash
    curl "https://api.rowbase.com/v1/datasets/ds_abc123" \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from rowbase import Rowbase

    client = Rowbase(api_key="YOUR_API_KEY")
    dataset = client.datasets.get("ds_abc123")

    print(f"Name: {dataset.name}")
    print(f"Rows: {dataset.row_count}")
    print(f"Columns: {[c.name for c in dataset.columns]}")
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const { Rowbase } = require('@rowbase/sdk');

    const client = new Rowbase({ apiKey: 'YOUR_API_KEY' });
    const dataset = await client.datasets.get('ds_abc123');

    console.log(`Name: ${dataset.name}`);
    console.log(`Rows: ${dataset.rowCount}`);
    ```
  </Tab>
</Tabs>

## Get Dataset Data

To retrieve the actual rows from a dataset, use the data endpoint:

```bash
curl "https://api.rowbase.com/v1/datasets/ds_abc123/data" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

See [Dataset Data](/api-reference/datasets/data) for full documentation.

## Errors

| Status | Code | Description |
|--------|------|-------------|
| 404 | `not_found` | Dataset doesn't exist |
| 403 | `authorization_error` | No access to this dataset |
