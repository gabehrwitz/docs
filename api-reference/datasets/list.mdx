description: 'Retrieve a list of datasets in your organization'

---

## Endpoint

```
GET https://api.rowbase.com/v1/datasets
```

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `project_id` | string | Filter by project |
| `limit` | integer | Results per page (default: 20, max: 100) |
| `cursor` | string | Pagination cursor |

## Example Request

<Tabs>
  <Tab title="cURL">
    ```bash
    curl "https://api.rowbase.com/v1/datasets?project_id=proj_xyz&limit=10" \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from rowbase import Rowbase

    client = Rowbase(api_key="YOUR_API_KEY")
    datasets = client.datasets.list(project_id="proj_xyz", limit=10)

    for dataset in datasets:
        print(f"{dataset.name}: {dataset.row_count} rows")
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const { Rowbase } = require('@rowbase/sdk');

    const client = new Rowbase({ apiKey: 'YOUR_API_KEY' });
    const datasets = await client.datasets.list({
      projectId: 'proj_xyz',
      limit: 10
    });

    datasets.data.forEach(ds => {
      console.log(`${ds.name}: ${ds.rowCount} rows`);
    });
    ```
  </Tab>
